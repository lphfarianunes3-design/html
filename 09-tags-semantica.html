tags semanticas em html 5

as tags semanticas são elementos em html que descrevem claramente o significa do conteudos que elas envolvem. Elas ajudam a melhorar a acessibilidade a seo e a estrutura do documento.aqui estão algumas das principais tags semantica

  • <header>: define o cabeçalho de um documento ou seção
  • <nav>: define um conjunto de links de navegação
  • <main>: define o conteudo principal do documento
  • < section >: define o conteudo principal do documento
  • < article >: define o conteudo independente e auto-contido cmo um artigode blog ou noticia
  • < aside >: define o conteudo relacionado mas separando do conteudo principal como uma barra lateral.
  • < footer >: define um conteudo ou seção

usar essas tags semantica ajuda os motores de busca a entender melhor o conteudo do seu site e melhora a experiencia do usuario especialmente para aqueles que utilizam leitores de tela.
